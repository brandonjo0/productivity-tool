## Getting Started

This is a simple command-line tool for studying.
Built with Python, Tkinter (for file selection), and Google Generative AI (Gemini API).

### Prerequisites

Using Python 3.9+, install the required packages using the following pip commands:
  ```sh
  python -m pip install -q -U google-genai
  python -m pip install python-dotenv
  python -m pip install PyMuPDF
  ```

### Installation

1. Get a free API Key at [https://aistudio.google.com/api-keys](https://aistudio.google.com/api-keys)
2. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
3. Create a new file called `.env`
4. Enter your API in `.env`
   ```js
   API_KEY = ENTER YOUR API KEY;
   ```
